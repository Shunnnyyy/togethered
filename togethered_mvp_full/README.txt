TogetherEd MVP Full

How to run:
1. Open this folder in PyCharm.
2. Open index.html in your browser.
3. Sign up or log in with any demo email.
4. The site uses localStorage, so posts and tutor requests persist in your browser.

Included pages:
- index.html
- login.html
- signup.html
- dashboard.html
- community.html
- tutors.html
- courses.html
- contact.html

Notes:
- This is a front-end MVP only.
- No real backend, database, or deployment is connected yet.
- Authentication is demo-only and stored in browser localStorage.
